# Kato 个人知识库 Demo

清新简约的个人知识库产品展示 Demo。

## 功能概览

- **新用户引导**：2 页渐显动画，身份输入必填，定制专属知识库
- **聊天区**：3 种工作模式（检索知识、新增知识、使用知识）可切换，提示词随模式变化
- **知识库**：6 个分类 + 增加更多卡片，2:3 比例，点击分类可进入详情
- **知识详情**：卡片展示（标题、摘要、主图/类型），按日期排序

## 技术栈

- Vue 3 + Vite
- 纯展示 + 聊天按钮基础交互
- 无后端，Mock 数据

## 运行

```bash
npm install
npm run dev
```

访问 http://localhost:5173

## 构建

```bash
npm run build
```
